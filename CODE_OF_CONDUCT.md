# Code of Conduct

This project follows the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/),
version 2.1.

Be respectful and constructive. Harassment, discrimination, and abusive behavior are not tolerated.
Report unacceptable behavior privately to **hi@zevs.gg**. Maintainers may remove, edit, or reject
contributions and may ban contributors for behavior they deem inappropriate.
